Privacy

Privacy policy

Ceiling is built to keep your AI usage data on your Windows PC. The public website uses limited, pseudonymous analytics so we can understand whether the site and download links are working.

The Ceiling desktop app

Ceiling reads usage from local app sessions and logs on your PC, or by calling the enabled provider's own usage endpoint. Credentials and usage data are stored only on your PC and are not sent to servers operated by Ceiling.

Usage requests go only to the provider you enabled or, for Wayfinder, the local gateway you configure. Ceiling does not require a Ceiling account.

The ceiling.win website

The website records page views and clicks on download and GitHub links through PostHog. These events can include the page path, referring site, browser user agent, and a pseudonymous visitor identifier derived from the request IP address and browser user agent. The identifier is salted and rotates every 30 days.

Ceiling's analytics code does not set advertising cookies, build advertising profiles, or sell personal information. Cloudflare hosts the website and processes requests as part of delivering and protecting it.

Third-party services

When you enable a provider in the desktop app, that provider handles requests under its own privacy policy. Downloads and source-code links may take you to GitHub. Website delivery and analytics use Cloudflare and PostHog respectively.

Your choices

You can stop website analytics requests with a browser content blocker. You can remove locally stored app data by uninstalling Ceiling and deleting its local application data. Provider-side data remains subject to that provider's controls and policies.

Contact

For privacy questions, open a request in the Ceiling GitHub issue tracker.
