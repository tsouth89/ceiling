Support

Help with Ceiling

Ceiling is an open-source Windows app. The fastest support path is the GitHub issue tracker, where reports can be followed through to a fix.

Report a problem or ask a question

Open a GitHub support request. Include your Ceiling version, Windows version, the provider involved, what you expected, and what happened. Remove tokens, cookies, email addresses, and other private information from screenshots and logs.

Install or update

Use the latest signed Windows installer. Existing installations can be updated by running the newer installer.

Data-source help

Read the data sources guide for how each provider is detected and what authentication method it uses.

Release history and source

See the latest release, browse the source code, or check existing issues before opening a new one.
